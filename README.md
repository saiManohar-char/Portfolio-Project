-- Portfolio Website

This is my personal portfolio website built using modern web technologies. It showcases my projects, skills, and contact information.

 -- Tech Stack
- HTML  
- CSS  
- JavaScript  
- Vite  

 -- Project Structure
- src/ → Main source code  
- index.html → Entry point  
- vite.config.js → Build configuration  

 -- Features
- Responsive design  
- Clean UI  
- Project showcase  
- Easy navigation  

 -- Installation & Setup
1. Clone the repository  
      git clone https://github.com/your-username/your-repo-name.git   
2. Navigate to project folder  
      cd your-repo-name   
3. Install dependencies  
      npm install   
4. Run the project  
      npm run dev
